# 欢迎来到 Glyphrun

Glyphrun 是 **AI 代理的应用商店** - 一个用于可运行数字资产 (RDAs) 的市场，我们称之为 **Glyphs**。

可以将它想象成一个应用商店，每个应用都是您可以通过 USDC 微支付即时运行的 AI 能力。

## 什么是 Glyph？

Glyph 是一个打包好的 AI 能力，您可以一键运行。Glyphs 有三种类型：

| 类型 | 描述 | 示例用例 |
|------|------|---------|
| **Prompts** | 基于 LLM 的文本生成 | 智能合约分析、代币命名、内容生成 |
| **Agents** | 通过 webhooks 的工作流编排 | 自动化交易、多步骤任务、数据管道 |
| **Tools** | 带有支付处理的 API 代理 | 区块链查询、社交分析、价格信息 |

## 分类

Glyphs 按结果分类：

- **LAUNCH** - 适合构建者：代币工程、社区管理、发布规划
- **SECURE** - 适合审计者：合约审计、身份验证、安全分析
- **OPTIMIZE** - 适合交易者：DeFi 策略、收益优化、市场分析

## 快速开始

三个简单步骤即可开始：

1. **连接钱包** - 使用 Privy 登录（邮箱、钱包或社交账号）
2. **充值账户** - 向您的钱包存入 USDC
3. **运行 Glyph** - 从市场中找到并执行任何 glyph

[开始使用](/zh/get-started)

## 适合创作者

构建并变现您自己的 AI 能力：

- 创建 Prompts、Agents 或 Tools
- 灵活定价（按次、按 token 或免费）
- 每次有人运行您的 glyph 时赚取 USDC
- 通过详细分析追踪性能

[创作者指南](/zh/guides/creators/creating-prompt-glyph)

## 使用 xPay 支付

Glyphrun 使用 [xPay](https://xpay.sh) 在 Base 网络上进行安全的 USDC 微支付：

- **即时** - 约 2 秒结算
- **透明** - 按使用付费
- **非托管** - 您控制您的资金
