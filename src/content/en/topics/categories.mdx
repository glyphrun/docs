# Categories

Glyphs are organized by outcome to help users find the right tools.

## The Three Categories

### LAUNCH

**For Builders** creating new projects.

#### Use Cases

- Token engineering and design
- Community strategy planning
- Launch checklist generation
- Branding and naming
- Technical documentation

#### Example Glyphs

- **Token Name Generator** - Creative name suggestions
- **Tokenomics Simulator** - Economic model analysis
- **Launch Checklist** - Step-by-step launch plan
- **Community Strategy** - Growth planning

#### Target Users

- Project founders
- Token creators
- Community builders
- Marketing teams

### SECURE

**For Auditors** and security professionals.

#### Use Cases

- Smart contract analysis
- Security vulnerability detection
- Identity verification
- Rug pull detection
- Risk assessment

#### Example Glyphs

- **Contract Analyzer** - Security vulnerability scan
- **Rug Pull Detector** - Scam warning system
- **Wallet Reputation** - Address risk scoring
- **Token Safety Check** - Contract red flags

#### Target Users

- Security auditors
- Risk analysts
- Due diligence teams
- Investors

### OPTIMIZE

**For Traders** and DeFi users.

#### Use Cases

- Yield comparison and optimization
- Market sentiment analysis
- Portfolio analysis
- Price monitoring
- Trading strategy development

#### Example Glyphs

- **Yield Comparator** - Best yield opportunities
- **Sentiment Analyzer** - Market mood tracking
- **Portfolio Analyzer** - Holdings breakdown
- **Price Alert** - Token price monitoring

#### Target Users

- DeFi users
- Traders
- Yield farmers
- Portfolio managers

## Category Selection

When creating a glyph, choose the category that best matches:

### Primary Outcome

What does the user achieve?

- **Building something** → LAUNCH
- **Assessing risk/security** → SECURE
- **Optimizing returns** → OPTIMIZE

### Target Audience

Who will use this glyph?

- **Builders/Creators** → LAUNCH
- **Auditors/Analysts** → SECURE
- **Traders/Investors** → OPTIMIZE

### Examples

| Glyph Idea | Category | Reasoning |
|------------|----------|-----------|
| Token name generator | LAUNCH | Creating new project assets |
| Contract vulnerability scanner | SECURE | Security assessment |
| APY comparison tool | OPTIMIZE | Yield optimization |
| Community growth planner | LAUNCH | Building community |
| Wallet address checker | SECURE | Risk assessment |
| Trading signal analyzer | OPTIMIZE | Trading improvement |

## Filtering by Category

Users filter the marketplace by category:

1. Go to [Explore](https://glyphrun.com/explore)
2. Select category filter
3. See relevant glyphs

### Multi-Category Glyphs

Some glyphs fit multiple categories. Choose the **primary** outcome:

- If it primarily helps build → LAUNCH
- If it primarily assesses risk → SECURE
- If it primarily optimizes returns → OPTIMIZE

## Category Icons

Each category has distinct visual styling:

- **LAUNCH** - Rocket/launch imagery
- **SECURE** - Shield/lock imagery
- **OPTIMIZE** - Chart/growth imagery
