# Run Your First Glyph

This guide walks you through running your first glyph on Glyphrun.

## Prerequisites

Before running a glyph:

1. [Create an account](/get-started/create-account)
2. [Fund your wallet](/get-started/fund-wallet) with USDC

## Step 1: Browse the Marketplace

1. Go to [Explore](https://glyphrun.com/explore)
2. Browse available glyphs or use filters:
   - **Category** - LAUNCH, SECURE, or OPTIMIZE
   - **Type** - Prompt, Agent, or Tool
   - **Price** - Free, under $0.05, under $0.10
   - **Verified** - Show only verified glyphs

## Step 2: Select a Glyph

Click on any glyph card to view its detail page:

- **Description** - What the glyph does
- **Pricing** - Cost per run
- **Stats** - Total runs, rating, success rate
- **Creator** - Who built this glyph

## Step 3: Configure Inputs

Each glyph has an input form based on its schema:

### For Prompts

1. Fill in the required text fields
2. Select a **Model Tier**:
   - Fast ($0.02) - Quick responses
   - Balanced ($0.05) - Best quality/speed
   - Reasoning ($0.10) - Complex analysis
3. Review the estimated cost

### For Agents/Tools

1. Fill in the required fields
2. Review the fixed price per run

## Step 4: Run the Glyph

1. Click the **Run** button
2. Confirm the transaction (first time only)
3. Wait for execution (typically 2-30 seconds)

## Step 5: View Results

Results appear in the console panel:

- **Prompts** - Markdown-formatted text output
- **Agents** - Status updates and final result
- **Tools** - JSON response data

### Actions

- **Copy** - Copy output to clipboard
- **Download** - Save as file
- **Share** - Share result link

## Example: Running a Prompt Glyph

Let's run the "Token Name Generator" glyph:

1. Go to [Token Name Generator](https://glyphrun.com/glyph/token-name-generator)
2. Fill in the form:
   - **Theme**: DeFi
   - **Style**: Professional
   - **Keywords**: yield, vault, secure
3. Select **Balanced** tier ($0.05)
4. Click **Run**
5. View generated token names in the output

## Viewing Run History

See all your past runs at [Account > History](https://glyphrun.com/account/history):

- Run ID and timestamp
- Glyph name and type
- Cost and status
- Link to view full output

## Troubleshooting

### "Insufficient Balance"

Your wallet doesn't have enough USDC:
- [Fund your wallet](/get-started/fund-wallet)
- Check for locked funds in pending transactions

### "Glyph Not Found"

The glyph may have been removed or unpublished:
- Try a different glyph from the [marketplace](https://glyphrun.com/explore)

### "Execution Error"

The glyph encountered an error during execution:
- Check your inputs are valid
- Try running again
- Contact support if the issue persists
