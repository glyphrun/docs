# Get Started

Welcome to Glyphrun! This guide will help you get up and running quickly.

## Prerequisites

Before you begin, you'll need:

- A web browser (Chrome, Firefox, Safari, or Edge)
- An email address or Web3 wallet for authentication
- USDC on Base network for running glyphs (optional for browsing)

## Quick Start Steps

### 1. Create an Account

Visit [glyphrun.com](https://glyphrun.com) and sign in using Privy:

- **Email** - Sign in with your email address
- **Wallet** - Connect MetaMask, Coinbase Wallet, or other Web3 wallets
- **Social** - Use Google, Twitter, or Discord

[Learn more about account creation](/get-started/create-account)

### 2. Fund Your Wallet

To run glyphs, you'll need USDC in your Glyphrun wallet:

- Deposit USDC from your external wallet
- Purchase USDC directly with a card (coming soon)
- Start with the default $10 spending allowance

[Learn more about funding](/get-started/fund-wallet)

### 3. Run Your First Glyph

Browse the marketplace and find a glyph to run:

1. Go to [Explore](https://glyphrun.com/explore)
2. Filter by category (LAUNCH, SECURE, OPTIMIZE)
3. Click on a glyph to see details
4. Fill in the required inputs
5. Click **Run** to execute

[Detailed walkthrough](/get-started/run-first-glyph)

## Understanding Glyphs

Before diving in, it helps to understand [what glyphs are](/get-started/what-are-glyphs) and the three types available:

- **Prompts** - LLM-based text generation with model selection
- **Agents** - Workflow automation via webhooks
- **Tools** - API proxies with authentication handling
